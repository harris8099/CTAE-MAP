# 🏫 CTAE Campus Map

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen.svg)](https://harris8099.github.io/CTAE-MAP/)
[![Documentation](https://img.shields.io/badge/Docs-Technical_Details-blue.svg)](docs/documentation.md)

This is an interactive campus map web application designed to help students, staff, and visitors navigate the college campus efficiently. Built with HTML, CSS, JavaScript, and the Leaflet.js library, the application provides detailed information about various departments, building floors, and key facilities.

![CTAE Campus Map Preview](https://github.com/user-attachments/assets/434d2cb4-0b4a-40f5-bcf0-bb81ccef6541)

## 🌟 Features

* **Interactive Campus Map:** Powered by Leaflet.js, navigate a fully interactive map with zoom, pan, and click functionality. 🗺️
* **Detailed Department Info:** View rich metadata for buildings, including descriptions, contact details, and reference images. 🏢
* **Floor-wise Navigation:** Drill down into specific buildings to see exactly which labs, offices, and rooms are located on each floor. 🏢➡️🏠
* **Smart Search:** Quickly find specific departments, hostels, or facilities using the built-in search bar. 🔍
* **Geolocation:** Instantly find your current location on the campus map with the click of a button. 📍

```bash

git clone [https://github.com/harris8099/CTAE-MAP.git](https://github.com/harris8099/CTAE-MAP.git)

# Navigate into the project directory:
